# Vienna PO Job Tracker

Auto-syncing Senior Product Owner job tracker for Vienna, Austria.
Built for Mrunal Moghe · Powered by Adzuna API + GitHub Actions + GitHub Pages.

## How it works

1. **GitHub Actions** fetches live PO roles from Adzuna every 6 hours
2. Results are saved to `jobs.json` in this repo
3. **GitHub Pages** serves `index.html`, which loads `jobs.json` on every visit
4. Your application statuses are saved in your browser across visits

## Setup (5 minutes)

### Step 1 — Get a free Adzuna API key

1. Go to [developer.adzuna.com](https://developer.adzuna.com/)
2. Click **Sign Up** → create a free account
3. Go to **Dashboard** → copy your `App ID` and `App Key`

### Step 2 — Add secrets to GitHub

1. In your GitHub repo → **Settings** → **Secrets and variables** → **Actions**
2. Click **New repository secret** and add:
   - `ADZUNA_APP_ID` → paste your App ID
   - `ADZUNA_APP_KEY` → paste your App Key

### Step 3 — Enable GitHub Pages

1. In your repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main`, folder: `/ (root)`
4. Click **Save**

### Step 4 — Run the first sync

1. Go to **Actions** tab in your repo
2. Click **Sync Vienna PO Jobs** → **Run workflow**
3. Wait ~30 seconds → `jobs.json` will appear in your repo
4. Your dashboard at `https://YOUR-USERNAME.github.io/vienna-po-tracker` now shows live roles

## Features

- Live roles fetched from Adzuna (Vienna, Product Owner / Senior PO)
- Direct links to each job posting
- Application status tracking saved in browser
- Add roles manually with custom URLs
- Filter by sector, status, date posted
- Sort by any column
- New roles since last visit highlighted with a **New** badge
- Auto-refreshes every 6 hours via GitHub Actions (free tier covers this)
- Dark mode support
